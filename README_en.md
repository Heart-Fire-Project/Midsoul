# Midnight Soul

> Last Edited: 2.0.0 | 2026/03/05  
> 点击[这里](https://github.com/Heart-Fire-Project/Midsoul/blob/main/README.md)来查阅中文版 

This is the datapack repository of the original ABA map by Heart Fire Project: **Midsoul**!

## How to Play

You can download the map directly from [Releases](https://github.com/Heart-Fire-Project/Midsoul/releases) or go [here](https://www.bilibili.com/read/cv46487047) to read the introduction (sry that it's Chinese) and download the map.

## Help Us

If you encounter any issues (or have any suggestions), feel free to share them in [Issues](https://github.com/Heart-Fire-Project/Midsoul/issues) or [Discussions](https://github.com/Heart-Fire-Project/Midsoul/discussions)!  
You can also join the [DC Group](https://discord.gg/jxUxX283Jd) for discussions.

### Contribute to Development

Please join the [DC Group](https://discord.gg/jxUxX283Jd) and consult the mods for furthur information.

### Multi-language (i18n)

English translation is now open on Paratranz. Please apply to become a member [here](https://paratranz.cn/projects/17920) using code `Midsoul-i18n`. At the same time, you are also welcome to localize the project into other languages on your own.
